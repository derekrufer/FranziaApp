# Account Rights Notes

Use this when building authentication and authorization.

## Commissioner Assignment

- Dom should be assigned the `Commissioner` role.

## Delegated Commissioner Rights

Derek and Mitch should have commissioner-level rights for these actions:

- Sync Fleaflicker rosters.
- Sync Fleaflicker traded picks.
- Upload/update rankings.
- Select/edit keepers for any team.

These rights do not necessarily need to make Derek and Mitch full commissioners for every admin action. Model them as granular permissions if possible, so the app can distinguish full commissioner access from delegated commissioner tools.
